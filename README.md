# ⚡ КликТохн Telegram Bot v1.0.1

Бот с системой кликов, аукционов, NFT-предметов, реферальной системой и премиум-подписками.

## 🚀 Быстрый старт на Railway

### 1️⃣ Подготовка
```bash
# Копируйте .env.example в .env и заполните значения
cp .env.example .env
```

### 2️⃣ Переменные окружения для Railway
- `BOT_TOKEN` — от @BotFather
- `OWNER_ID` — ваш ID в Telegram
- `DB_NAME` — `/data/clicktohn.db` (для Volume)

### 3️⃣ Git & Deploy
```bash
git push railway main
```

**Подробное руководство:** см. [RAILWAY_SETUP.md](RAILWAY_SETUP.md)

---

## 📡 Архитектура

```
main.py              → точка входа + фоновые задачи
config.py            → конфигурация (версия, токен, параметры игры)
database.py          → работа с SQLite БД
keyboards.py         → клавиатуры для пользователя
states.py            → FSM состояния
handlers/             → обработчики команд  
  ├─ admin.py        → /admin команды владельца
  ├─ user.py         → профиль, кредиты
  ├─ shop.py         → магазин улучшений
  ├─ chat.py         → игра "найди чат"
  ├─ pvp.py          → боевая система
  ├─ trade.py        → торговля между игроками
  ├─ nft.py          → система NFT
  ├─ history.py      → история игрока
  └─ common.py       → общие команды
```

---

## 📊 Базовые команды

- `/start` — начать игру
- `/profile` — мой профиль
- `/shop` — магазин усилений
- `/auction` — аукцион NFT
- `/trade` — торговля игроками
- `/admin` — для владельца бота (OWNER_ID)

## 🔧 Локальное тестирование

```bash
# Установите зависимости
pip install -r requirements.txt
```

## 💾 База данных

## 🎮 Основные системы

### Система кликов
- Базовая мощность: 0.05 💢
- Пассивный доход, улучшения силы клика
- Пакеты кликов за рубли

### NFT Аукционы
- 15 уровней редкости (сафари 0.010-10% шанс)
- Мин. 2, макс. 25 участников
- День аукциона, победитель получает NFT

### Реферальная система
- +200 💢 за первого рефа
- +100 💢 за каждого последующего

### VIP/Premium пакеты
- 2x/3x множители кликов и дохода
- На неделю, месяц или навсегда
